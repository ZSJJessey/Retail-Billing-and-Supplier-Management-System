# Retail-Billing-and-Supplier-Management-System
技术栈： SpringBoot、SpringMVC、MyBatis、Thymeleaf、MySQL、Redis、Java、Interceptor、Prompt Engineering、LLM 应用原型

• 完成零售账单管理系统的需求分析、数据库建模与后端开发，覆盖账单、供应商、用户、密码修改及文件管理等核心业务模块 。

• 基于 SpringBoot + MyBatis 搭建分层架构，支持多条件查询、信息维护、登录鉴权与页面交互，形成可演示的完整业务闭环 。

• 结合 Redis 缓存与拦截器机制优化系统访问性能和登录状态管理，提升系统基础稳定性与可维护性 。

• 在项目复盘阶段补充智能化扩展方案，围绕账单检索、供应商问答、运营摘要生成等场景设计离线 Demo 思路，形成传统管理系统 AI 应用演进的实践认知。
